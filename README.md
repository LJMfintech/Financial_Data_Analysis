# Financial_Data_Analysis - 금융데이터분석 1차 과제
## <과제 간단 요약 및 접근방법>

**1. 10개 국가에 대해 WRDS에서 월별 기업 단위 데이터 수집**

   -> WRDS에서 데이터 가져오는 코드 구현

**2. 기업 단위 월별 수익률, 시가총액, 국가 단위 월별 수익률 계산**

   -> 시장총액와 월별 수익률을 계산하기 위해 변수들을 먼저 구하고, (조정주가, 수익률 등)
   
   -> 그것을 토대로 시장가치와 수익률을 구하고, 구한 값들을 토대로 수익률 데이터셋을 새로 생성한 후,

   -> 그래프를 이용하여 데이터 시각화 진행
   

**3. 코로나 시기와 이후 회복 시기 구분하기**

   -> 2번에서 구한 수익률 데이터셋에

   -> 2020.03.01 ~ 2021.12.31 (코로나 기간) / 2022.01.01 ~ 2024.12.31 (회복 시기)로 기간을 나눈 다음

   -> 'period' 칼럼을 따로 추가해서 나눈 기간을 구분하여 넣음
   


**4. 각 국가의 기간별로 수익률의 기초통계량 계산**

   -> 구하고자 하는 기초통계량들을 함수로 정의한 다음, 각 국가의 기간과 수익률에 기초통계량 함수를 적용하여 계산
   


**5. 선진국과 신흥국의 비교**

   -> 10개 나라를 선진국 (developed)과 신흥국 (emerging)으로 분류한 다음, 그룹간 수익률의 기초통계량을 구한 뒤 비교

   -> 분류기준: 교수님이 분류한 기준표
   


**6. 기간별, 국가별 수익률의 상관계수 행렬 구하기 및 결과 비교**

   -> 기간별, 국가별로 수익률의 상관계수 행렬을 구한 다음 결과를 통해 비교, 분석 진행
   


**7. 총정리**

   -> 국가별 (10개 국가), 기간별 (코로나 기간/회복시기), 그룹별 (선진국/신흥국)로 통계량을 토대로 각 분류별 주요 차이점 위주로 서술
